# go-test
Unit testing in Go

Recommended Assertion package
`github.com/stretchr/testify/assert`

To run all tests
`go test`

To run just one test
`go test -run TestNameOfTest`

Coverage Report
`go test -coverprofile cover.out`
`go tool cover -html=cover.out -o cover.html`
